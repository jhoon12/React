# 이벤트 헨들링

이벤트란?

사용자가 웹 브라우저에서 DOM요소들과 상호작용 하는 것을 이벤트라고 합니다.

## 1.onChange 이벤트 핸들링하기

```react
//EventPractice.js
import React, {Component} from 'react';

class EventPractice extends Component{
    render(){
        return (
            <div>
                <h1>이벤트 연습</h1>
                <input
                    type="text"
                    name="message"
                    placeholder="아무거나 입력해 보세요"
                    onChange={
                        (e) => {
                            console.log(e);
                        }
                    }
                >
                </input>
            </div>
        );
    }
}

export default EventPractice;
```

실행 시 SyntheticEvent라는 객체를 볼 수 있다. e객체는 네이티브 이벤트를 감싸는 객체이다.

## state에 input값 담기

```react
import React, {Component} from 'react';

class EventPractice extends Component{
    state = {
        message:''
    }
    render(){
        return (
            <div>
            <h1>이벤트 연습</h1>
            <input
            type="text"
            nams="message"
            placeholder="아무거나 입력해 보세요"
            value={this.state.message}
            onChange={
                (e)=>{
                    this.setState({
                        message:e.target.value
                    })
                }
            }
            ></input>
            </div>
        );
    }
}

export default EventPractice;
```

## input 여러 개 다루기



사실 메소드를 여러 개 만들어서 할 수도 있다. 하지만 event객체를 이용하면 훨씬 간단하다.

```react
import React, {Component} from 'react';

class EventPractice extends Component{
    state = {
        message:'',
        username:''
    }
    handleChange = (e)=>{
        this.setState({
            [e.target.name]: e.target.value
        });
    }
    handleClick = (e)=>{
        alert(this.state.message+ ':' +this.state.username);
        this.setState({
            message:'',
            username:''
        })
    }
        render(){
        return (
            <div>
                <h1>이벤트 연습</h1>
                <input
                type="text"
                name="message"
                placeholder="아무거나입력해요"
                value={this.state.message}
                onChange={this.handleChange}
                />
                <input
                type="text"
                name="username"
                placeholder="사용자명"
                value={this.state.username}
                onChange={this.handleChange}>
                </input>
                <button onClick={this.handleClick}>
                    정답
                </button>
            </div>
        );
    }
}

export default EventPractice;
```

이 코드의 핵심은 이 부분이다.

```react
handleChange = (e)=>{
        this.setState({
            [e.target.name]: e.target.value
        });
    }
```

객체 안에서 key를 []로 감사면 그 안에 넣은 레퍼런스가 가리키는 실제 값이 key값으로 사용된다.

즉 e.target,name은 input의 name을 가리키고 있는 것이다.

정리하면 [e.target.name] = "username" 이므로 setState로 같은 이름의 key인 username이 바뀐다.



## onKeyPress 이벤트 핸들링

```react
import React, {Component} from 'react';

class EventPractice extends Component{
    state = {
        message:'',
        username:''
    }
    handleChange = (e)=>{
        this.setState({
            [e.target.name]: e.target.value
        });
    }
    handleClick = (e)=>{
        alert(this.state.message+ ':' +this.state.username);
        this.setState({
            message:'',
            username:''
        })
    }
    handleKeyPress = (e) =>{
        if(e.key === 'Enter'){
            this.handleClick();
        }
    }
        render(){
        return (
            <div>
                <h1>이벤트 연습</h1>
                <input
                type="text"
                name="message"
                placeholder="아무거나입력해요"
                value={this.state.message}
                onChange={this.handleChange}
                onKeyPress={this.handleKeyPress}
                />
                <input
                type="text"
                name="username"
                placeholder="사용자명"
                value={this.state.username}
                onChange={this.handleChange}>
                </input>
                <button onClick={this.handleClick}>
                    정답
                </button>
            </div>
        );
    }
}

export default EventPractice;
```
